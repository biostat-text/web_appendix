R のダウンロードとインストール方法

1. R とは？

R は、統計解析やデータ可視化に広く使われている無料のソフトウェア（オープンソース）です。R 本体をインストールすると、コマンドを入力して解析できるようになります。

3. ダウンロード（CRAN から）

R は CRAN（Comprehensive R Archive Network）という公式の配布サイトから入手できます。

  1. CRAN のトップへアクセス：https://cran.r-project.org/
  2. 使っている OS（Windows / macOS / Linux）を選びます
  3. 表示される案内に従ってインストーラ（またはパッケージ）を入手します

4. Windows の手順

  1. CRAN で Download R for Windows をクリック
  2. base をクリック
  3. Download R x.y.z for Windows（最新版）をクリックしてインストーラ（.exe）を保存
  4. 保存した .exe をダブルクリックして起動
  5. 基本的には「次へ」を押して進めてOK（インストール先は標準のままで問題ありません）
  6. 完了後、「スタート」メニューから R を起動できます

5. macOS の手順

  1. CRAN で Download R for macOS をクリック
  2. お使いの macOS バージョンに合った .pkg（インストーラ）を選び、ダウンロード
  3. .pkg を開き、画面の案内に従ってインストール
  4. インストール後、アプリケーションから R を起動できます
